MathSnap — Privacy Policy

Last updated: 19 August, 2025

MathSnap is a lightweight browser extension that converts LaTeX math formulas into PNG images. This privacy policy explains what (if any) data the extension collects, how it is used, and what control you have over it.

Data Collection

MathSnap does not collect, store, or share personal information about its users. The extension performs LaTeX rendering and PNG export locally within your browser.

Local Rendering & Clipboard

All LaTeX-to-PNG rendering is performed locally by the extension code included in the package. The extension requests the clipboardWrite permission so that users can copy rendered images or LaTeX code to their clipboard when they choose. MathSnap does not read data from the clipboard.

Optional AI Assist (User-controlled)

MathSnap offers an optional AI Assist feature that converts plain-text math or informal notation into LaTeX. This feature is disabled by default. If you enable it, MathSnap will send conversion requests directly to the Google Gemini API endpoint (https://generativelanguage.googleapis.com/*) using the API key you provide. These API requests are made from your browser to Google and are not proxied or stored by MathSnap's developer servers.

Third-Party Services

No analytics, advertising, telemetry, or third-party tracking services are included in MathSnap. The only external network requests possible are those you explicitly trigger by enabling the optional AI Assist feature.

Security

MathSnap does not execute remote JavaScript or WebAssembly. All functionality is delivered within the extension package and does not rely on loading external scripts at runtime.

User Control & Data Removal

Because MathSnap does not collect or store personal data, there is no user data retained by the extension developer to remove. If you wish to revoke network access for AI Assist, disable the feature in the extension options and remove any API keys you stored locally.
