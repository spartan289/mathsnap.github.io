MathSnap — Privacy Policy

Last updated: 30 August, 2025

MathSnap is a lightweight browser extension that converts LaTeX math formulas into PNG images. This privacy policy explains what data the extension collects, how it is used, and what control you have over it.

Data Collection

MathSnap does not collect, store, or share personal information about its users. The extension performs LaTeX rendering and PNG export locally within your browser.

Local Rendering & Clipboard

All LaTeX-to-PNG rendering is performed locally by the extension code included in the package. The extension requests the clipboardWrite permission so that users can copy rendered images or LaTeX code to their clipboard when they choose. MathSnap does not read data from the clipboard.

Optional AI Assist (User-controlled)

MathSnap offers an optional AI Assist feature that converts plain-text math or informal notation into LaTeX. This feature is disabled by default. If you enable it, MathSnap will send your input text or image to MathSnap’s secure backend service.

No personal data is stored by MathSnap’s servers. Requests are processed in real-time and discarded after conversion.

Third-Party Services

No analytics, advertising, telemetry, or third-party tracking services are included in MathSnap. The only external network requests possible are those triggered when you use the AI Assist feature, which are sent first to MathSnap’s backend.

Security

MathSnap does not execute remote JavaScript or WebAssembly. All functionality is delivered within the extension package. AI Assist requests are transmitted securely (HTTPS) and processed without long-term storage.

User Control & Data Removal

You can enable or disable the AI Assist feature at any time in the extension options. Since MathSnap does not store personal data, there is no user data retained that requires removal.
